##  Projeto principal

###  My Accountability Buddy

Está sendo projeto para ser um app fullstack focado em organização pessoal (finanças, fitness, nutrição e produtividade). Atualmente estamos implementando o backend e banco de dados da aplicação, posterioemente a ideia é implementar uma versão mobile (React Native) e uma versão web (React)

O foco aqui não foi só “fazer funcionar”, mas estruturar bem:

- Separação por domínios (cada parte do sistema bem definida)
- Controle de histórico (não perder evolução dos dados)
- Modelagem de banco pensada pra escalar
- Backend estruturado com Prisma + PostgreSQL

👉 https://github.com/Ricardo-9/accountability-buddy-backend
( Consultar branch DEV - Possui o conntéudo atualizado da aplicação)
